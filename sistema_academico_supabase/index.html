No sistema acadêmico, os alunos poderão consultar suas notas e frequências a qualquer momento, de forma rápida e online, através do portal disponível na plataforma escolar.



A SisAcademy, em parceria com os estudantes do curso técnico de Desenvolvimento de Sistemas da Proz Educação,idealizou este site como um trabalho acadêmico, com o objetivo de facilitar o acompanhamento do desempenho escolar, garantindo que os estudantes possam verificar se estão alcançando a média e a frequência mínima exigidas para aprovação.



Aqui podemos cadastrar os professoresa e os alunos matriculados:


Professores

Após o cadastramento, os professores poderão facilmente lançar as notas e registrar a frequência dos alunos de forma prática e eficaz.

Alunos

Após o cadastro, os alunos poderão verificar as suas notas e frequência, acompanhando de perto o seu desempenho.



Importante:

  • Nota média para aprovação: 7,0. Essa média é alcançada através da média aritmética das 4 notas bimestrais.

  • Frequência para aprovação:75%. Faltas justificadas através de requisição apresentada pelo aluno serão abonadas.

    • O conselho escolar é soberano para definir a aprovação mesmo que os resultados não tenham sido alcançados.
